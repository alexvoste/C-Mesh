## Description


C-Mesh Accessor is a tool for bypassing blocks and smartly routing internet traffic.


The browser extension works in conjunction with a proxy server (Node.js or Cloudflare Workers), allowing you to proxy only selected sites (Telegram, YouTube, Discord, etc.) or all traffic.


**How it works:**

- Browser → PAC script → Proxy → Target server

- If the proxy is blocked, simply change the link in the settings without reinstalling the extension


**Key features:**

- Minimalist interface

- Selective domain-based routing

- Support for wildcard patterns (*.example.com)

- Settings saving

- Backend built in pure JavaScript using Factory Pattern
